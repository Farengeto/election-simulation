How to use the input file:

Parties:
Party definitions consist of three lines
First line is the name of the party.
Second line defines the party's approval rating in %.
Third line defines the party's colours in RGB.

Provinces and Regions:
Provinces are defined by their name, delimited by a ':'
Subsequent regions are placed in the respective province.
The first column of region declarations defines the name
The second column of region declarations defines the population
The third column of region declarations defines the seat count
Each subsequent column defines the party's support in the region.
Support levels are unitless. They may consist of and total any value.
